# Project management

## Description
Each user should be able to create testing projects and manage its own projects

## Requirements
- Each project should have name, and tags
- There is a many to many relation between user and project
- Admin is the one that can grant access of a project to other users
- Project name and tags can be edited
- There is delete, we archive projects
- Admin can see and unarchive projects
**Note: Admin is django admin**

## Constraints
- Remove the dummy sidebar items and put projects for now
- For create and edit project, use modal
- Projects should have pagination
- We should have filter by tags, and search ability
