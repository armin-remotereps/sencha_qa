# Upload test cases

## Description
User should be able to upload testrails exported xml to our system

## Requirements
- We should have an upload area so user can upload it's file
- User should be able to drag and drop the file
- User should be able to see uploaded file and its processing status
- User should be able to stop processing of a file
- User should be able to filter test cases based on uploaded file
- User should be able to delete an uploaded file, and all of that file test cases should be deleted

## Constraints
- Try to have a component for drag and drop for reusability
- Check the file structure before uploading it so won't waste server resources
- Use web socket for live update of file status
- The file should be processed on celery
- Delete the file after processing, we don't need the file itself
- For all deleting don't forget to confirm it with user
