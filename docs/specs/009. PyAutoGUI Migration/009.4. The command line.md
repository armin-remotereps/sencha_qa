# The command line

## Description
We want to be able to execute shell commands on the client as well

## Requirements
- We need ability to run shell command on client beside the py_auto_gui
- The client will receive shell commands from web socket and will execute it on the remote machine, then will send it back to the server
- The consumer need to have actions for running command so channel layer can be call it

## Verification
We need to run the client and a management command to send client a command for doing an action, like echo hello world, and the result should be the hello world
